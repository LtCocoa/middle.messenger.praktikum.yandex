### Установка
```
npm install
```
### Сборка
```
npm run build
```
### Запуск сервера разработки
```
npm start-parcel
```
### Запуск express сервера
```
npm run start-server
```
### Запуск линтера
```
npm run lint
```
### Запуск линтера с исправлением ошибок
```
npm run lint-fix
```
